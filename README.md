# LeadFlow

LeadFlow is a SaaS-style platform designed to manage leads,
campaigns and commercial opportunities with an analytical dashboard.

## Tech Stack

### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL

### Frontend
- React
- Vite
- React Router
- Reactstrap

## Features

- Lead management
- Campaign tracking
- Groups organization
- Analytical dashboard
- Collapsible sidebar navigation

## Future Improvements

- Authentication (JWT)
- Role-based access control
- AI scoring service (Python)
- Notification service (Java)
